    이벤트 타겟 : 이벤트가 발생하는 대상 ex ) input 태그 , button 태그 등

    이벤트 설치방식.
     1) inline : 
     2) 프로퍼티리스너 :
      var t = document.getElementById('#id');
      t.onclick= function(event){
        console.log(event);
      }
     3) addeventlistener
      t.addeventlistener('click', function(event){
        alert('Hello world, '+event.target.value);
      });
